# 🚀 75 Days LeetCode Challenge  

A 75-day commitment to mastering **Data Structures & Algorithms** through consistency, discipline, and structured problem-solving.

---

## 📌 About  

This repository documents my journey of solving LeetCode problems daily to strengthen DSA fundamentals and improve coding efficiency.

Each solution focuses on:
- Clean and readable C++ code  
- Optimized approach  
- Strong understanding of core concepts  
- Time & Space complexity awareness  

---

## 🎯 Objectives  

- Build strong problem-solving skills  
- Master common DSA patterns  
- Improve coding speed and accuracy  
- Prepare for technical interviews  
- Develop daily coding discipline  

---

## 🛠 Tech Stack  

- **Language:** C++  
- **Platform:** LeetCode  
- **Core Topics:**  
  Arrays, Strings, Hashing, Sorting, Two Pointers, Recursion, and more.
